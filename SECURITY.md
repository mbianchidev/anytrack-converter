# Security Policy

## Supported Versions

We provide security updates for the latest version on the default branch.

## Reporting a Vulnerability

Please report security issues privately using GitHub Security Advisories:
https://github.com/mbianchidev/anytrack-converter/security/advisories/new

If you cannot access that page, open a minimal issue asking for a security contact.

## Response Timeline

- Acknowledgement within 48 hours
- Initial assessment within 5 business days
- Coordinated disclosure within 90 days when possible

## Security Best Practices

- Avoid uploading sensitive files.
- Keep FFmpeg and yt-dlp updated in your environment.
- Use HTTPS when deploying publicly.
